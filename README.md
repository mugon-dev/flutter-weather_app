# weather_app
https://openweathermap.org/current <br/>
https://openweathermap.org/api/air-pollution

api 연동을 통해 json 형식으로 데이터를 받아와서 도시, 시간, 온도, 날씨에 따른 아이콘 변경, 대기오염 지수에 따른 아이콘 변경, 미세먼지, 초미세먼지 구현

## dependencies
